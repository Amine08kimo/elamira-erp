  SI PROBLÈME

Erreur "Erreur de chargement du module Maintenance" ?
  → Exécuter : python nettoyer_cache.py
  → Puis : python main.py

KPIs vides (tout à 0) ?
  → Exécuter : python tools\load_maintenance_demo.py
  → Puis : python main.py

Module ne s'affiche pas ?
  → Vérifier : test_maintenance_import.py
  → Lire : RESOLUTION_ERREUR_MAINTENANCE.md
